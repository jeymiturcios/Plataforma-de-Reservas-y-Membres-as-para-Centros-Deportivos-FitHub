//FitHub - Plataforma de Reservas Deportivas

Proyecto backend desarrollado con Node.js, Express y PostgreSQL.

//Funcionalidades

- Registro de usuarios
- Creación de reservas
- Validación de reservas duplicadas
- Control de cupo en actividades
- Manejo de errores

//Tecnologías

- Node.js
- Express
- PostgreSQL
- Nodemon
